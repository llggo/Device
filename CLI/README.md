#Device CLI
Run arg -c | --com <com name | number>
Command Line:
-s | --show, hiện led
	Option: 
		-a | --address <number>, địa chỉ led
		-t | --Text	<string>, nội dung hiện
		-r | --run, bool chữ chạy

-o | --on 
	Option:
		-a | -address <number>, địa chỉ led

-f | --off 
	Option:
		-a | -address <number>, địa chỉ led
